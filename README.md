# Grokking Algorithms

A comprehensive repository for learning and implementing **Data Structures and Algorithms (DSA)** from the book **"Grokking Algorithms"** by Aditya Bhargava.

## 📚 Overview

This repository contains implementations of fundamental algorithms and data structures covered in "Grokking Algorithms." Each chapter focuses on a specific algorithmic concept with practical implementations in multiple programming languages (Python, Java, C++).

## 📖 Chapters & Topics

### Chapter 1: Introduction to Algorithms
- **Binary Search** - Efficient searching in sorted arrays
  - `binary_search.py`
  - `binary_search.java`

### Chapter 2: Selection Sort
- **Selection Sort** - A simple sorting algorithm with O(n²) time complexity
  - `selection_sort.py`
  - `selection_sort.java`
  - `selection_sort.cpp`

### Chapter 3: Recursion
- Understanding recursive problem-solving approaches
  - `call_stack.py` - Understanding how call stacks work
  - `look_for_key.py` - Recursive search example

### Chapter 4: Quicksort
- **Quicksort** - Efficient divide-and-conquer sorting algorithm
  - `quick_sort.py`
  - `quick_sort.java`
  - `quick_sort.cpp`
  - `sum_array.py` - Recursive array sum
  - `count_array.py` - Recursive array counting

### Chapter 5: Hash Tables
- **Hash Tables** - Key concepts and practical applications
  - `check_voter.py` - Preventing duplicate votes
  - `get_page.py` - Caching web pages

## 🎯 Purpose

This repository is designed for:
- **Beginners** learning DSA concepts with clear, intuitive explanations
- **Students** studying algorithms with hands-on implementations
- **Practitioners** revisiting fundamental concepts
- **Interview Preparation** - Common algorithms asked in technical interviews

## 🛠️ Getting Started

### Prerequisites
- Python 3.x
- Java JDK
- C++ compiler (g++)

### Running Python Scripts
```bash
python3 filename.py
```

### Compiling and Running Java
```bash
javac filename.java
java filename
```

### Compiling and Running C++
```bash
g++ -o output filename.cpp
./output
```
